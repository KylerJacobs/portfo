001 : Password Security

This password checker uses a .txt file to read what the users password is. Running this program in the command prompt will save the users password, in memory, on the machine. Uploading the password into a .txt file instead,and then having the program read the .txt file, ensures the users password is not saved anywhere after the user is done.
